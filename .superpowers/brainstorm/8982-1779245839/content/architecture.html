Diseño completo — Google Photos → Europa page

Tres fases: setup único antes del viaje, sync automático durante, y display en la página.

Fase 1 — Setup (una vez, antes del viaje)

scripts/setup-google-photos-auth.ts — corre localmente una sola vez
1. Crear proyecto en Google Cloud
2. Habilitar Photos Library API
3. Crear OAuth2 credentials
4. Correr script → abre browser
5. Autorizar acceso a tus fotos
✅ Script guarda REFRESH_TOKEN
Vercel env vars (una vez):
GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET
GOOGLE_REFRESH_TOKEN
SYNC_SECRET (para proteger el endpoint)

Fase 2 — Sync automático (cada noche durante el viaje)

Flujo del cron — medianoche hora local de cada ciudad
⏰ Vercel Cron
Todos los días a las 23:00
GET /api/europa/sync-photos
+ SYNC_SECRET header
→
📷 Google Photos API
1. Detecta ciudad actual por fecha
2. Fetch fotos del día (mediaItems:search)
3. Score por metadata:
• Elimina duplicados (<5seg)
• Prioriza mayor resolución
• Distribuye por hora del día
4. Selecciona top 10
→
🗄 Supabase
5. Descarga imágenes → Storage
/europa-photos/paris/2026-06-12/
6. Guarda metadata en tabla:
city_slug, taken_date
public_url, width, height
display_order, score
